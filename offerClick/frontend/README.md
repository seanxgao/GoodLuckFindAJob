# OfferClick Frontend

React + TypeScript frontend for the OfferClick job application assistant.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The UI will be available at `http://localhost:5173`

## Build

To build for production:
```bash
npm run build
```

## Keyboard Shortcuts

- `J` / `K`: Navigate up/down through job list
- `Enter`: Select/open job
- `A`: Apply to selected job
- `S`: Skip selected job
- `Shift + S`: Star/unstar selected job

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Zustand (state management)
- React Query (data fetching)
- react-hotkeys-hook (keyboard shortcuts)
- Lucide React (icons)
